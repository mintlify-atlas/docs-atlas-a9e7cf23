# Factorial Example

This example demonstrates how to calculate the factorial of a number using an iterative approach with a `while` loop in Volette.

## What is Factorial?

The factorial of a number `n` (written as `n!`) is the product of all positive integers less than or equal to `n`. For example:
- `5! = 5 × 4 × 3 × 2 × 1 = 120`
- `3! = 3 × 2 × 1 = 6`

## Implementation

Here's the complete factorial implementation in Volette:

```rs
fn factorial(n: i32): i32 {
    let result = 1
    let i = 1
    while i <= n {
        result *= i
        i += 1
    }
    return result
}
```

### How It Works

1. **Initialize variables**: Start with `result = 1` (the factorial accumulator) and `i = 1` (the loop counter)
2. **Loop condition**: Continue while `i <= n`
3. **Multiply**: In each iteration, multiply `result` by `i` using the `*=` operator
4. **Increment**: Increase `i` by 1 to move to the next number
5. **Return**: After the loop completes, return the final result

This iterative approach is efficient and avoids the overhead of recursive function calls.

## Building and Running

<Steps>
  <Step title="Create the Volette file">
    Save the factorial function to a file:
    
    ```bash
    # subjects/factorial.vt
    fn factorial(n: i32): i32 {
        let result = 1
        let i = 1
        while i <= n {
            result *= i
            i += 1
        }
        return result
    }
    ```
  </Step>

  <Step title="Compile to assembly">
    Use the Volette compiler to generate assembly code:
    
    ```bash
    cargo run -- build subjects/factorial.vt -o subjects/factorial.s
    ```
  </Step>

  <Step title="Assemble the object file">
    Convert the assembly to an object file:
    
    ```bash
    as -o subjects/factorial.o subjects/factorial.s
    ```
  </Step>

  <Step title="Create a C wrapper">
    Create a C program to call the Volette function:
    
    ```c
    // subjects/main.c
    #include <stdio.h>
    
    extern int factorial(int n);
    
    int main() {
        printf("factorial(5) = %d\n", factorial(5));
        printf("factorial(7) = %d\n", factorial(7));
        printf("factorial(10) = %d\n", factorial(10));
        return 0;
    }
    ```
  </Step>

  <Step title="Link and execute">
    Compile the C wrapper and link with the Volette object file:
    
    ```bash
    gcc subjects/main.c subjects/factorial.o -o subjects/factorial
    ./subjects/factorial
    ```
    
    Expected output:
    ```
    factorial(5) = 120
    factorial(7) = 5040
    factorial(10) = 3628800
    ```
  </Step>
</Steps>

## Key Language Features

This example demonstrates several Volette features:

- **Type annotations**: `n: i32` specifies the parameter type, `: i32` specifies the return type
- **Variable declarations**: Using `let` to declare variables
- **While loops**: Iterating with a condition-based loop
- **Compound assignment**: Using `*=` and `+=` operators
- **Return statements**: Explicit return of the final value
