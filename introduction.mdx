# Welcome to Volette

Volette is an **expression-oriented, statically-typed programming language** that compiles to native code. It's designed to be simple and straightforward, with a focus on a powerful type system.

Built in Rust, Volette uses a heavily modified [Citadel](https://github.com/Isible/citadel) backend to generate efficient assembly code.

<Note>
Currently, Volette only supports **aarch64 macOS**. Support for additional platforms is planned for future releases.
</Note>

## Quick Example

Here's a simple Volette program that calculates the Fibonacci sequence:

```volette
fn fib(n: i32): i32 {
    let a = 0
    let b = 1
    let i = 0
    while i < n {
        let tmp = a + b
        a = b
        b = tmp
        i += 1
    }
    return a
}
```

## Why Volette?

Volette combines the best of modern language design:

- **Simple & Clean**: Optional semicolons, expression-based syntax, and minimal boilerplate
- **Type Safe**: Comprehensive type system with type inference where you need it
- **Fast Compilation**: Written in Rust for quick compile times
- **Native Code**: Direct compilation to assembly for optimal runtime performance

<Note>
Volette is in active development. APIs and language features may change as the project evolves.
</Note>
